# Khushal Mehta — Portfolio

Portfolio built with **Next.js** and **Tailwind CSS**, showcasing my projects and résumé.

## Local Development

```bash
git clone https://github.com/Khushal-Me/Portfolio.git
cd Portfolio
pnpm install
pnpm dev   # http://localhost:3000
```
